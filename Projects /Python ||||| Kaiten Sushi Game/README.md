## 📝 게임 개요

- **게임 목표**  
  레일 위의 접시 가운데 빈 공간에 '방향키 이벤트'를 수행함으로써 초밥을 채워넣는 간단한 게임입니다.  
  방향키 입력을 성공적으로 수행해 초밥을 완성하고 금액을 획득하세요!

- **게임 규칙**
  - **Game Over**: 방향키 이벤트를 연속으로 실패하면 게임 종료.  
  - **승리 조건**: 일정 금액 이상을 획득하면 승리합니다!
